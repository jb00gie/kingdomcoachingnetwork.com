Forces Users to Login to view the site.

== Description ==

If you are not logged into the site it automatically redirects you to the login screen.
All you have to do is activate the plugin to turn it on.  Deactivate to turn it off.
After some testing this plugin seems to work fine with multisite

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
